Псевдокласс :not в jQuery

Псевдокласс not исключает те элементы из найденных, которые соответствуют указанному селектору.

Давайте в следующем задании получим и покрасим в красный цвет все заголовки НЕ с классом zzz:

$(':headers:not(.zzz)').css('color',
'red');
